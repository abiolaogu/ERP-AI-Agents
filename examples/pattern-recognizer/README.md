# Pattern Recognizer AI Agent

Anomaly detection and correlation discovery.

## Features
- Anomaly detection (Z-score, IQR, Isolation Forest)
- Correlation discovery
- Trend analysis
- Pattern recognition

## Quick Start
```bash
pip install -r requirements.txt
uvicorn app:app --host 0.0.0.0 --port 8098
```

**Version**: 1.0.0
