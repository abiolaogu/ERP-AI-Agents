# Image Restorer AI Agent

AI-powered image upscaling, colorization, and restoration.

## Features
- Image upscaling (up to 8K)
- Colorization
- Denoising
- AI-based restoration

## Quick Start
```bash
pip install -r requirements.txt
uvicorn app:app --host 0.0.0.0 --port 8104
```

**Version**: 1.0.0
