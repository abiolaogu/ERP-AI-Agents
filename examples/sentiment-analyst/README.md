# Sentiment Analyst AI Agent

Multi-lingual emotion detection and sentiment analysis.

## Features
- Multi-lingual support (20+ languages)
- Emotion detection
- Sentiment classification
- Key phrase extraction

## Quick Start
```bash
pip install -r requirements.txt
uvicorn app:app --host 0.0.0.0 --port 8099
```

**Version**: 1.0.0
