# Educational Tutor AI Agent

Adaptive learning and Socratic questioning for personalized education.

## Features
- Adaptive learning paths
- Socratic questioning
- Personalized curriculum
- 10K+ concurrent students

## Quick Start
```bash
pip install -r requirements.txt
uvicorn app:app --host 0.0.0.0 --port 8095
```
